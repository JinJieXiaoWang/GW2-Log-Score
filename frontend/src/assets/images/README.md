# 图片资源目录

此目录用于存放项目的图片资源文件。

## 目录结构建议

```
images/
├── icons/          # 图标文件
├── logos/          # Logo文件
├── backgrounds/    # 背景图片
├── avatars/        # 头像图片
└── README.md       # 说明文档
```

## 图片格式建议

- **图标**: SVG 格式（可缩放、体积小）
- **照片**: WebP 或 JPEG 格式
- **复杂图形**: PNG 格式（支持透明）

## 注意事项

1. 图片文件命名使用小写字母和连字符，如 `hero-banner.png`
2. 对于图标优先考虑使用 SVG 或字体图标库
3. 大图片建议进行压缩处理
4. 避免提交过大的图片文件（超过 500KB 建议使用外部 CDN）
