# 评分系统规则说明

## 1. 评分系统概述

本系统基于Elite Insights日志数据，对玩家在WvW和PVE中的表现进行综合评分。评分范围为0-100分，分数越高表示表现越好。

## 2. WvW模式评分规则

### 2.1 职业定位

- **DPS**：输出职业，如战士、潜行者、元素使等
- **SUPPORT**：辅助职业，如守护者、魂武者等
- **UTILITY**：功能职业，如工程师、幻术师等

### 2.2 评分权重

#### DPS职业
- 输出评分：60%
- 倒地贡献：20%
- 生存评分：20%

#### SUPPORT职业
- 稳固覆盖：20%
- 抗性覆盖：15%
- 急速覆盖：15%
- 净化评分：20%
- 增益剥离：20%
- 生存评分：10%

#### UTILITY职业
- 增益剥离：30%
- 净化评分：30%
- 破控评分：30%
- 生存评分：10%

### 2.3 计算方法

1. **输出评分**：基于玩家DPS与团队平均DPS的比值，归一化到0-100分
2. **倒地贡献**：基于玩家倒地次数与团队平均倒地次数的比值，归一化到0-100分
3. **生存评分**：基础100分，每倒地一次扣20分，每死亡一次扣50分
4. **稳固/抗性/急速覆盖**：基于Buff覆盖时间百分比，直接转换为0-100分
5. **净化/剥离评分**：基于每分钟净化/剥离次数与团队平均值的比值，归一化到0-100分
6. **破控评分**：基于每分钟破控次数，归一化到0-100分

## 3. PVE模式评分规则

### 3.1 职业定位

- **DPS**：输出职业
- **SUPPORT**：辅助职业

### 3.2 评分权重

#### DPS职业
- 输出评分：60%
- 破控评分：20%
- 生存评分：20%

#### SUPPORT职业
- 稳固覆盖：30%
- 抗性覆盖：30%
- 关键Buff覆盖：40%（急速或敏捷的最高值）

### 3.3 计算方法

1. **输出评分**：基于玩家DPS与团队平均DPS的比值，归一化到0-100分
2. **破控评分**：基于玩家破控伤害与团队平均破控伤害的比值，归一化到0-100分
3. **生存评分**：基础100分，每倒地一次扣20分，每死亡一次扣50分
4. **稳固/抗性覆盖**：基于Buff覆盖时间百分比，直接转换为0-100分
5. **关键Buff覆盖**：基于急速或敏捷的最高覆盖时间百分比，直接转换为0-100分

## 4. 评分异常问题解决方案

### 4.1 问题描述

- 部分用户显示96分但无详细数据
- 部分用户显示42.16分但无详细数据

### 4.2 原因分析

1. **数据缺失**：可能是因为日志数据中某些字段缺失或格式错误
2. **计算异常**：可能是因为分母为0导致的计算错误
3. **数据类型问题**：可能是因为数据类型转换错误

### 4.3 解决方案

1. **前端数据处理**：在PlayerDetail组件中添加数据验证和默认值处理
2. **后端数据计算**：在ScoringEngine中添加异常处理和边界条件检查
3. **数据存储优化**：确保所有评分数据完整存储

## 5. 评分等级说明

- **95-100分**：优秀
- **90-94分**：良好
- **85-89分**：中上
- **80-84分**：中等偏上
- **75-79分**：中等
- **70-74分**：中等偏下
- **60-69分**：及格
- **50-59分**：需改进
- **30-49分**：较差
- **0-29分**：极差

## 6. 数据来源

评分数据来源于Elite Insights生成的战斗日志，包含以下核心数据：

- 玩家DPS
- 破控伤害
- 倒地次数
- 死亡次数
- Buff覆盖时间
- 净化次数
- 增益剥离次数
- 倒地贡献

## 7. 注意事项

1. 评分系统仅作为参考，不代表玩家的全部实力
2. 不同职业和定位的评分标准有所不同，请合理看待
3. 评分结果可能受到团队配置、战斗持续时间等因素影响
4. 建议结合详细数据综合评估玩家表现
