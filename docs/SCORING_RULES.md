# 评分规则管理

本文档合并了评分系统规则说明和评分规则管理规范。

## 评分系统概述

基于Elite Insights日志数据，对玩家在WvW和PVE中的表现进行综合评分。评分范围为0-100分。

## WvW模式评分规则

### 职业定位
- **DPS**: 输出职业
- **SUPPORT**: 辅助职业
- **UTILITY**: 功能职业

### 评分权重

#### DPS职业
- 输出评分：60%
- 倒地贡献：20%
- 生存评分：20%

#### SUPPORT职业
- 稳固覆盖：20%
- 抗性覆盖：15%
- 急速覆盖：15%
- 净化评分：20%
- 增益剥离：20%
- 生存评分：10%

#### UTILITY职业
- 增益剥离：30%
- 净化评分：30%
- 破控评分：30%
- 生存评分：10%

### 计算方法
1. **输出评分**: 基于DPS与团队平均比值
2. **倒地贡献**: 基于倒地次数与平均比值
3. **生存评分**: 基础100分，倒地扣20分，死亡扣50分
4. **Buff覆盖**: 基于时间百分比
5. **净化/剥离**: 基于每分钟次数
6. **破控评分**: 基于每分钟破控次数

## 评分规则管理规范

### 设计目标
- **统一管理**: 所有评分规则集中存储在 `config/scoring_rules.json`
- **职业-专精级别**: 支持按职业、专精、姿态、武器组合定制评分
- **可调整**: 游戏版本更新时只需修改配置文件
- **可扩展**: 易于添加新的评分维度和规则

### 架构
```
scoring_rules.json (唯一真实来源)
         │
         ├──> 后端 ScoringRuleLoader / ConfigLoader
         │            │
         │            ├──> WvWRuleScorer (基于规则的评分器)
         │            │
         │            └──> API /api/scoring/*
         │
         └──> 前端 (预留)
```

### 数据结构
包含version, mode, default_threshold, dimension_definitions, profession_specialization_rules等。

### 核心字段说明
- version: 规则版本
- mode: 游戏模式 (WvW/PVE)
- default_threshold: 默认阈值
- dimension_definitions: 评分维度定义
- profession_specialization_rules: 职业专精规则
- fallback_role_rules: 后备角色规则
- squad_size_rules: 团队规模规则
- normalization_rules: 归一化规则

### 规则加载流程
1. 启动时加载scoring_rules.json
2. 验证规则完整性
3. 构建评分器实例
4. 缓存规则数据

### 动态配置
支持运行时更新规则配置。

### 规则更新流程
1. 修改scoring_rules.json
2. 重启服务验证
3. 测试评分准确性
4. 文档同步更新

## 评分异常问题解决方案

### 问题描述
- 部分用户显示96分但无详细数据
- 部分用户显示42.16分但无详细数据

### 原因分析
1. **数据缺失**: 日志数据字段缺失或格式错误
2. **计算异常**: 分母为0导致的计算错误
3. **数据类型问题**: 数据类型转换错误

### 解决方案
1. **前端数据处理**: 在PlayerDetail组件中添加数据验证和默认值处理
2. **后端数据计算**: 在ScoringEngine中添加异常处理和边界条件检查
3. **数据存储优化**: 确保所有评分数据完整存储

## 评分等级说明

- **95-100分**: 优秀
- **90-94分**: 良好
- **85-89分**: 中上
- **80-84分**: 中等偏上
- **75-79分**: 中等
- **70-74分**: 中等偏下
- **60-69分**: 及格
- **50-59分**: 需改进
- **30-49分**: 较差
- **0-29分**: 极差

## 数据来源

评分数据来源于Elite Insights生成的战斗日志，包含以下核心数据：
- 玩家DPS
- 破控伤害
- 倒地次数
- 死亡次数
- Buff覆盖时间
- 净化次数
- 增益剥离次数
- 倒地贡献

## 注意事项

1. 评分系统仅作为参考，不代表玩家的全部实力
2. 不同职业和定位的评分标准有所不同，请合理看待
3. 评分结果可能受到团队配置、战斗持续时间等因素影响
4. 建议结合详细数据综合评估玩家表现
