\# GW2日志解析项目规则（仅本项目生效） 1. 硬性要求：新增/修改功能\*\*必须Python后端+Vue3前端同时完成\*\*，禁止单端开发 2. 技术栈固定：FastAPI + Vue3 + PrimeVue + Tailwind CSS，禁止其他UI/样式方案 3. 后端：按api/service/model/utils分层，接口完整、注释清晰 4. 前端：用PrimeVue组件+Tailwind原子类，保持UI统一 5. 前后端闭环：接口→页面一一对应，提交必同步 6. 代码全量中文注释，方便后续维护\
\# 项目规则补充（UI风格） 1.  统一卡片样式：使用圆角、浅背景、柔和阴影，和当前「今日战场统计」卡片保持一致 2.  统一主色调：沿用当前的紫色主色，按钮、高亮文字用对应色值，禁止新增其他高饱和颜色 3.  统一文字排版：标题用稍大字体+深灰，正文用中灰，辅助文字用浅灰，保持层级清晰 4.  统一间距：所有组件、卡片、按钮之间的间距保持一致，使用Tailwind类名控制
